# 霓虹跑酷 3D (Neon Parkour 3D)

這是一個以 HTML、CSS 和 JavaScript 打造的 3D 霓虹風格跑酷遊戲。玩家需要在充滿科幻感的賽道上躲避障礙物，挑戰最高分數！

## 🎮 線上遊玩

你可以點擊下方連結直接在瀏覽器中遊玩：
**[👉 點此遊玩 霓虹跑酷 3D](https://as955076as-netizen.github.io/neon-parkour/)**

## 🕹️ 遊戲操作

* **電腦版**：使用鍵盤 `A` / `D` 鍵或 `左` / `右` 方向鍵來左右移動。
* **手機版**：點擊螢幕左側或右側來控制角色移動。

## 🛠️ 技術說明

* 單一 `index.html` 檔案架構。
* 使用原生的 HTML5 Canvas 或 WebGL 技術渲染 3D 畫面。
* 響應式設計，支援電腦與手機版面。
* 無需安裝任何依賴，直接開啟檔案即可執行。

## 🚀 部署說明

本專案使用 GitHub Pages 進行免費託管。只要將程式碼推送到 GitHub 儲存庫的 `main` 分支，並在 Settings -> Pages 中開啟 GitHub Pages 功能即可自動部署。
